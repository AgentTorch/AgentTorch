# Experiment Formats

| Format | Data Source | Usage |
|--------|-------------|--------|
| exp1 (default) | Processed binary skill values | `EXPERIMENT_ID = "exp1"` in mipro_skills.py |
| exp2 | Original O-NET work activities + skills strings | `EXPERIMENT_ID = "exp2"` in mipro_skills.py |

Both use same train/test splits, only input representation differs.
